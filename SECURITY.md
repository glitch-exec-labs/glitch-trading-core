# Security Policy

## Reporting A Vulnerability

If you believe you have found a security issue, configuration leak, credential exposure, or other sensitive vulnerability related to this repository, please report it privately:

- Email: support@glitchexecutor.com

Please include:

- the affected repository and file or component
- a short description of the issue
- reproduction steps or a proof of concept if safe to share
- the potential impact

Please do not open a public issue for undisclosed security vulnerabilities.
