# Social Preview Brief

This brief defines the GitHub social preview image for Glitch Trading Core.

## Purpose

Glitch Trading Core should read as the ecosystem hub.

The image should communicate:

- umbrella architecture
- premium engineering posture
- multi-repo family coordination
- MT5 today and cTrader next

## Export Spec

- Size: `1280 x 640`
- Format: `PNG`
- Safe area: keep all important text inside the left `60%` of the canvas
- Style: dark, premium, technical, restrained

## Exact Text

- Title: `GLITCH TRADING CORE`
- Subtitle: `Umbrella Architecture`
- Tagline: `Shared modules, Oracle coordination, and platform direction for MT5 today and cTrader next.`
- Footer strip: `Hub of the Glitch Ecosystem`

## Layout

- Place the title in the upper-left quadrant
- Put the subtitle directly under the title with tight spacing
- Set the tagline in one or two short lines below the subtitle
- Reserve the right half for an abstract systems visual
- Place the footer strip along the bottom edge

## Visual Direction

- Use a near-black background with subtle graphite gradients
- Add a faint grid or market-structure texture
- Build an abstract network diagram with one central core and several connected strategy branches
- Suggest platform portability with geometric execution blocks rather than broker logos

## Palette

- Background: near-black and graphite
- Primary accent: emerald
- Secondary accent: muted steel
- Optional highlights: soft cyan for node glows

## Typography

- Title: bold, condensed, uppercase or strong title case
- Subtitle: smaller uppercase with wider tracking
- Tagline: clean sans-serif, readable at thumbnail size

## Avoid

- no screenshots
- no code snippets
- no cluttered dashboards
- no literal broker logos
- no more than one visual focal point on the right side
