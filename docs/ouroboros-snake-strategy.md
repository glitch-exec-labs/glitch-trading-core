# Ouroboros Snake Strategy

## Positioning

Ouroboros Snake Strategy is the flagship Glitch ensemble strategy.

It is not just a random collection of bots. It is the coordinated execution stack formed by:

- Oracle as the coordination and conflict-resolution layer
- Viper
- Cobra
- Taipan
- Mamba
- Anaconda
- Hydra

## Why The Name Fits

The Ouroboros brand works well because the strategy is cyclical, self-reinforcing, and adaptive:

- multiple strategy forms feed one coordinated portfolio process
- the Oracle governs how independent signals become portfolio actions
- the ensemble can evolve without losing the core identity

## Internal Structure

### Coordination layer

- Oracle
- portfolio risk governance
- conflict resolution
- exposure shaping

### Execution layer

- Viper for fast trend continuation
- Cobra for structure-aware price action
- Taipan for session breakout logic
- Mamba for mean reversion
- Anaconda for slower breakout confirmation
- Hydra for regime-aware adaptation

## What Is Not Ouroboros

The following should be treated as adjacent strategy products, not as the core Ouroboros stack:

- Indian King Cobra
- Terciopelo

They can be part of the broader Glitch ecosystem, but they should keep their own product identities and repositories.

## Recommended One-Line Description

Ouroboros Snake Strategy is Glitch's flagship multi-bot ensemble, combining Oracle coordination with the six-snake execution stack.
